# Contra Galleries MGMT

Management dashboard/internal tool used for a private art gallery based in Chelsea, NYC. 
Fully managed artist, artwork and exhibition profiles, with API endpoints for a separate public-facing React application to pull data from.

Future releases will also include QR code generation.

Key tools used; NextJS, MUI, JSX, Supabase.
